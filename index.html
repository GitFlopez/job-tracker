📋 How to Use the Job Tracker

Quick guide to get you started

🚀 Quick Start

  1. JD Inbox: Select source → Paste job descriptions → Click "Analyze Batch" for AI scoring
  2. Job Leads: High-match positions (Tier 1) auto-route here → Move to Tracker when ready to apply
  3. Active Tracker: Track applications, outcomes, and follow-ups
  4. Settings: Configure industries, salary thresholds, tech stack, and AI preferences

📊 Understanding Tiers

  • Tier 1 (⭐⭐⭐⭐⭐): Salary ≥$110K + Target Industry → Auto-routes to Job Leads
  • Tier 2 (⭐⭐⭐): Salary ≥$110K OR Target Industry → Stays in Inbox for review
  • Tier 3 (⭐): Below thresholds → Consider archiving

💡 Pro Tips

  • Use Manual Entry for quick single job adds
  • Use Batch Paste for analyzing multiple JDs at once
  • Export CSV regularly for backups (weekly recommended)
  • Configure Settings → Industry Rankings to customize tier scoring
  • Enable AI Analysis in Settings → API Configuration for smarter recommendations
  • Check System Settings for auto-archive and language options

💾 Backup Reminder: Your data is stored locally in your browser. Export backups regularly to avoid data loss!
